# Cypress

# Run all tests
> npx cypress run --browser chrome --headed --spec "cypress/e2e/*.cy.js"

### Cypress features
1. Page object models
2. fixtures
3. mocking (cy.intercept)
4. network requests (cy.request)
5. cypress config
6. visual testing [cypress-image-diff](https://github.com/haim-io/cypress-image-diff)
7. screenshots
8. videos
9. ci github actions
