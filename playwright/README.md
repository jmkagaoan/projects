# Playwright

### Run all tests on all browsers
> npx playwright test

### Run all ui tests on just chrome
> npx playwright test --project=chromium

### Run all api tests
> npx playwright test --project=api

### Playwright features
1. storageState
2. project dependencies
3. parallelism
4. page object models
5. fixtures
6. playwright config
7. videos
8. screenshots
9. ci github actions
